# Container Loading Planner

Strumento web per la pianificazione del carico in container marittimi. Disposizione automatica, calcolo baricentro, distribuzione assi, impilamento, report PDF.

## Quickstart

Vedi `DEPLOY.md` per la guida passo-passo al deploy gratuito (Vercel + Supabase, 30 minuti).

## Stack

- HTML/JS vanilla, niente build pipeline
- Supabase per auth + database (free tier)
- Vercel per hosting (free tier)

## Local development

Apri `public/index.html` in un browser. Per testare il login devi prima configurare Supabase (vedi DEPLOY.md).
